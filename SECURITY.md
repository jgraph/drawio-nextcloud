# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the draw.io Nextcloud integration, please report it responsibly by opening a [private security advisory](https://github.com/jgraph/drawio-nextcloud/security/advisories/new) on GitHub.

Please do **not** open a public issue for security vulnerabilities.

## Scope

This policy covers the Nextcloud integration code in this repository (PHP backend, JavaScript frontend, configuration handling). The draw.io editor itself is maintained separately at [jgraph/drawio](https://github.com/jgraph/drawio) â€” please report draw.io editor vulnerabilities there.

## SOC 2

This repository is **not** covered by the JGraph SOC 2 process. We do not provide commercial services or support for this app.
